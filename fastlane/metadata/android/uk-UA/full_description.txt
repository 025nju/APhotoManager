<p>Особливості:</p>
<ul>
<li>Швидко знаходьте фотографії за тегами(ключові слова), (під-)папку, дату, географію, назву, ...</li>
<li>Відображення результатів у галереї або географічній карті з openstreetmap.</li>
<li>Функція детального перегляду збільшення масштабу, переміщення до наступного/попереднього зображення.</li>
<li>Вбудований файловий менеджер для фотографій: пошук, сортування, перегляд, копіювання, видалення, надсилання ... .</li>
<li>Редагування метаданних exif: дати, заголовку, опису, тегів(ключових слів), географії, рейтингу, ....</li>
<li>Автоматична обробка на основі правил-папки: фотографії автоматично перейменовуються та отримують теги, географічні назви, назви ... коли ви копіюєте/переміщуєте їх
*Позначте зображення як "ПРИВАТНЕ", щоб приховати їх від інших галереї-додатків та збирачів зображень.</li>
<li>У "Захищеному/закріпленому режимі" потрібні небезпечні команди для додатків, такі як редагування/копіювання/видалення/спільне використання/налаштування/"зміни вибору зображення" вимкнено, щоб ви могли передати телефон комусь іншому</li>
<li>Можна обробляти великі колекції зображень (20000+ зображень в 1000+ папках).</li>
<li>Використовує контент-провайдер зображення для Android. Немає необхідності в скануванні початкового зображення.</li>
<li>Розширений медіа-сканер для фото Exif, IPTC, XMP</li>
</ul>
<p>Необхідні дозволи для Android:</p>
<ul>
<li>INTERNET: для завантаження данних карт з OpenStreetmap Server</li>
<li>ACCESS_NETWORK_STATE and ACCESS_WIFI_STATE: щоб дізнатись, чи доступний Wi-Fi / Інтернет, щоб розпочати завантаження геоданих</li>
<li>WRITE_EXTERNAL_STORAGE для кешування завантажених даних карти в локальній файловій системі та для виконання файлів з фотографіями</li>
<li>READ_LOGS щоб читати та зберігати потенційні аварійні журнали в текстовому файлі.
.</li>
</ul>
