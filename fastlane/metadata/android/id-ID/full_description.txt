<p>Fitur:</p>
<ul>
<li>Cepat cari foto dengan tag (kata kunci), (sub-) folder, tanggal, geo, judul, ...</li>
<li>Menampilkan hasil di galeri atau peta geografis dari openstreetmap.</li>
<li>Detail melihat fitur zooming, gesek untuk gambar berikutnya / sebelumnya.</li>
<li>Buildin file manager untuk foto: temukan, sort, view, copy, delete, send, ....</li>
<li>Edit exif metadata: tanggal, judul, deskripsi, tag (kata kunci), geo, rating, ....</li>
<li>Dapat menangani koleksi gambar besar (20000+ gambar dalam 1000+ folder).</li>
<li>Menggunakan penyedia konten gambar Android. Tidak ada pemindaian gambar awal yang diperlukan.</li>
<li>Extended photo media scanner untuk Exif, IPTC, XMP</li>
</ul>
<p>Izin Android yang Diperlukan:</p>
<ul>
<li>INTERNET: untuk mendownload data peta dari Open Streetmap Server</li>
<li>ACCESS_NETWORK_STATE dan ACCESS_WIFI_STATE: untuk mengetahui apakah wifi / internet online untuk memulai geodata yang telah didownload</li>
<li>WRITE_EXTERNAL_STORAGE untuk menyimpan data peta yang diunduh di sistem file lokal dan melakukan operasi file dengan foto</li>
<li>READ_LOGS untuk membaca dan menyimpan potensi crash-log ke textfile.
.</li>
</ul>
