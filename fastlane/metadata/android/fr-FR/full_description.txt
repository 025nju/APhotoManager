<p>Fonctionnalités :</p>
<ul>
<li>Trouvez rapidement vos photos par mot-clé, (sous-)dossier, date, géolocalisation, titre ...</li>
<li>Affichez les résultats dans la Galerie ou une carte géographique d'OpenStreetMap.</li>
<li>Les fonctions d'affichage permettent de zoomer, de faire un glisser afin passer aux images suivantes ou précédente.</li>
<li>Gestionnaire de fichiers intégré pour les photos: trouver, trier, afficher, copier, supprimer, envoyer...</li>
<li>Éditez les métadonnées EXIF : date, titre, description, tag(mot-clé), géolocalisation, évaluation...</li>
<li>Peut manipuler de grandes collections d'images (+ de 20000 images dans + de 1000 dossiers).</li>
<li>Utilise le gestionnaire d'image natif d'Android. Aucune initialisation nécessaire.</li>
<li>Scan étendu d'images pour les formats EXIF, IPTC, XMP</li>
</ul>
<p>Permissions Android nécessaires:</p>
<ul>
<li>INTERNET : Pour télécharger les données cartographiques à partir du serveur OpenStreetMap</li>
<li>ACCESS_NETWORK_STATE et ACCESS_WIFI_STATE : pour savoir si le Wi-Fi/Internet est activé pour commencer à télécharger des  données cartographiques.</li>
<li>WRITE_EXTERNAL_STORAGE : pour mettre en cache les données cartographiques téléchargées dans le système de fichiers local , ceci dans le but d’effectuer certaines opérations plus rapidement.</li>
<li>READ_LOGS : Afin de lire et d'enregistrer des traces dans un fichier texte en cas de crache potentiels l'application.
.</li>
</ul>
