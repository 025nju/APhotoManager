<p>Argazki lokalak kudeatzeko argazki-galeria:</p>
<p>Ezaugarriak:</p>
<ul>
<li>Aurkitu azkar argazkiak etiketak(hitz-gakoak), (azpi)karpetak, data, informazio geografikoa, izenburua... erabiliz</li>
<li>Bistaratu emaitzak galerian edo mapa geografiko batean, openstreetmap-i esker.</li>
<li>Fitxategi-kudeaketa: Kopiatu, mugitu, aldatu izena, bidali, argazkien babes-kopia inkrementala ... .</li>
<li>Bilduma birtualentzako euskarria.</li>
<li>Editatu exif metadatuak: Data, izenburua, deskripzioa, etiketak(hitz-gakoak), geo, balorazioa, ....</li>
<li>Auto-prozesatzea:  Lekuz aldatzean aldatu izena automatikoki argazkia edo fitxategia  eta/edo gehitu metadatuak</li>
<li>Irudien bilduma handiak kudeatu ditzake (20000+ argazki 1000+ karpetatan).</li>
<li>Memorian arina (1.5 MB), memoria gutxiko Android 4.0 zaharretan dabil</li>
<li>Erabili Androiden irudi eduki-hornitzailea. Ez dira zertan irudiak aurretik eskaneatu behar</li>
<li>Exif, IPTC, XMP multimedia eta argazki eskaner hedatua</li>
</ul>
<p>Pribatutasuna:</p>
<ul>
<li>Argazkiak ezkutatu ditzake beste galeria edo irudi hautatzaileen begietara.</li>
<li>Gordailu modua: Gaituz gero nahi ez diren argazkiak ezin dira ikusi eta ekintza arriskutsuak (adib. ezabatu) debekatuta daude. Honela lasai utzi ahal  diozi zure gailua beste inori.</li>
<li>Argazkiak gailu lokalean geratzen dira. Ez dira inora igotzen.</li>
<li>Iragarkirik ez, jarraipen-elementurik ez, librea eta kode irekikoa, F-Droid biltegian eskuragarri</li>
</ul>
<p>Beharrezko Android baimenak:</p>
<ul>
<li>INTERNET: OpenStreetmap zerbitzaritik mapa deskargatzeko</li>
<li>ACCESS_NETWORK_STATE eta ACCESS_WIFI_STATE: wifi/internet ote dagoen antzemateko, geo datuak deskargatzen hasteko</li>
<li>WRITE_EXTERNAL_STORAGE deskargatutako mapak fitxategi sistema lokalean cache gisa gordetzeko eta argazkiekin eragiketak egiteko</li>
<li>READ_LOGS kraskatze txostenak (balude) irakurtzeko eta testu-fitxategian gordetzeko.</li>
</ul>
