<p>Características:</p>
<ul>
<li>Encontrar rápido fotos por etiquetas(palabras claves), (sub-)carpeta, fecha, mapa geográfico, título, ...</li>
<li>Mostrar resultado en galería o mapa geográfico desde el mapa de calle público.</li>
<li>Vista detallada características del enfoque, deslizar para imagen siguiente/anterior.</li>
<li>Creando el manejador de archivos para fotos: encontrar, ordenar, ver, copiar, eliminar, enviar, ... .</li>
<li>Editar metadata de exif: fecha, título, descripción, etiquetas(palabras claves), geo, valoración, ....</li>
<li>Puede manejar grandes colecciones de imágenes (20000+ imágenes en 1000+ carpetas).</li>
<li>Usa el proveedor de contenido de imágenes de Android. No es necesario un escaneo inicial de imágenes.</li>
<li>Escaneo extendido de multimedia de imágenes para Exif, IPTC, XMP</li>
</ul>
<p>Permisos requeridos de Android:</p>
<ul>
<li>INTERNET: para descargar el mapa de datos del servidor público de Streetmap</li>
<li>ACCESS_NETWORK_STATE y ACCESS_WIFI_STATE: para determinar si esta conectado al  wifi/internet para comenzar a descargar la datos geográficos</li>
<li>WRITE_EXTERNAL_STORAGE para guardar los datos del mapa descargado en el sistema de archivos local y realizar operaciones de archivos con las fotos</li>
<li>READ_LOGS para leer y salvar los registros de fallas potenciales en un archivo de texto.
.</li>
</ul>
